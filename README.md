# 应用接口签名(application interface signature)
为了保证数据在通信时的安全性，我们可以采用数据签名的方式来进行相关验证。

在设计签名算法的时候，主要考虑的是这几个问题:
* 请求参数是否被篡改；
* 请求来源是否合法；
* 请求是否具有唯一性。

详细实现见[WIKI](http://wenxinnnnn.com/)
